# FractaVolta Seconde Vie

**FractaVolta Seconde Vie** est une offre en construction pour les installations photovoltaïques existantes ou arrivant en fin de contrat.

L’idée est de ne pas laisser ces actifs solaires vieillir, être sous-valorisés ou retomber dans une simple logique d’injection abstraite. Ils peuvent devenir des nœuds locaux de capacité : stockage, usages agricoles, recharge, froid, eau, secours, compute local, batteries mobiles.

## Principe

```text
installation solaire existante
  -> audit technique et contractuel
  -> reprise ou accompagnement d’exploitation
  -> maintenance organisée
  -> stockage local si pertinent
  -> non-injection par défaut quand c’est utile
  -> réseau comme secours explicite
  -> revenu simple pour le propriétaire
  -> réversibilité organisée
```

## Pour les propriétaires

La promesse doit rester simple :

> Votre installation solaire entre dans sa seconde vie. Nous l’exploitons ou l’organisons avec des partenaires qualifiés. Vous gardez un revenu lisible, sans tracas excessifs, avec une sortie réversible.

FractaVolta ne promet pas une rente miraculeuse. L’objectif est plus sobre : faire mieux que l’alternative faible, éviter les soucis techniques, maintenir l’actif en état, et garder la valeur solaire au plus près du territoire.

## Réversibilité

La sortie doit être prévue dès le départ.

En cas de résiliation, FractaVolta retire ou désactive ses équipements propres, remet les données utiles, et coordonne la transition avec l’entité qui prend le relais.

Principe :

> Pas de pénalité cachée. La sortie est organisée, pas sanctionnée.

Les frais de sortie doivent être limités aux coûts techniques directs, nécessaires et justifiés.

## Marchés concernés

- hangars agricoles en fin d’obligation d’achat ;
- toitures professionnelles déjà équipées ;
- ombrières existantes ;
- petites centrales en seconde vie ;
- champs photovoltaïques existants pouvant être mieux territorialisés ;
- sites où une batterie locale ou mobile crée une vraie valeur.

## Ce que FractaVolta apporte

- doctrine de transition vers l’énergie-paquet ;
- structuration de l’offre ;
- recherche de sites pilotes ;
- coordination avec installateurs et mainteneurs ;
- logique de stockage local ;
- traçabilité des flux ;
- modèles contractuels réversibles ;
- lien vers des usages plus avancés : batteries mobiles, compute local, inférences garanties.

## Ce que FractaVolta ne remplace pas

FractaVolta ne remplace pas les installateurs certifiés, les bureaux de contrôle, les obligations Enedis, Consuel ou assurantielles.

La logique est partenariale : FractaVolta organise le modèle, les partenaires qualifiés sécurisent les opérations techniques.
