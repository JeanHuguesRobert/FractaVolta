# Partenaires installateurs

FractaVolta ne remplace pas les installateurs photovoltaïques, les mainteneurs, les bureaux d’études ou les acteurs du stockage.

Au contraire : FractaVolta cherche des partenaires techniques capables d’aider à transformer une doctrine en sites pilotes réels.

## Ce que FractaVolta apporte

- positionnement : transition vers l’énergie-paquet ;
- recherche de propriétaires et sites pilotes ;
- structuration commerciale : seconde vie, réversibilité, non-captation ;
- couche logicielle de suivi et de traçabilité ;
- modèles d’usage : stockage local, batteries mobiles, recharge, froid, eau, compute ;
- coordination entre propriétaire, installateur, mainteneur et usager local.

## Ce que le partenaire apporte

- audit technique ;
- sécurité électrique ;
- conformité ;
- estimation des coûts ;
- installation ou modification des équipements ;
- maintenance ;
- retour d’expérience terrain ;
- capacité à dire ce qui est faisable, coûteux, risqué ou inutile.

## Sites pilotes recherchés

- hangars agricoles en fin de contrat ;
- toitures professionnelles ;
- ombrières ;
- installations avec stockage existant ou possible ;
- sites agricoles ayant des usages utiles : froid, pompage, recharge, atelier ;
- bâtiments publics ou communaux.

## Première mission possible

Une première mission peut rester limitée :

```text
1. visiter un site
2. identifier la situation électrique et contractuelle
3. estimer l’état de l’installation
4. chiffrer une option de stockage local
5. identifier les contraintes Enedis / Consuel / assurance
6. proposer un scénario réversible
```

## Position claire

FractaVolta ne vient pas vendre une boîte magique.

FractaVolta vient proposer un nouveau marché : **opérer la seconde vie solaire locale comme capacité territoriale**, avec des partenaires certifiés pour la partie technique.

## Contact recherché

FractaVolta cherche quelques installateurs ou mainteneurs corses prêts à discuter d’un pilote sans engagement lourd.
