# Audit Seconde Vie

## Préqualifier avant de transformer

L’Audit Seconde Vie est la porte d’entrée de l’offre FractaVolta Seconde Vie.

Il sert à déterminer si un site peut devenir un nœud de capacité locale : énergie DC, batteries, usages critiques, inférence IA éventuelle, recharge de véhicules, coopérative ou participation habitante.

## Sites concernés

- centrale photovoltaïque ancienne ;
- hangar agricole ou artisanal ;
- toiture solaire existante ;
- ancienne centrale au sol ;
- friche rurale ;
- ancien vignoble ;
- terrain de montagne ;
- station-service ;
- dépôt ou site logistique.

## Données nécessaires

Pour une première préqualification :

- localisation ;
- photos ;
- surface ;
- statut du terrain ou du bâtiment ;
- puissance photovoltaïque existante ou envisagée ;
- production connue, si disponible ;
- contrat d’achat ou factures, si disponibles ;
- accès routier ;
- usages locaux possibles ;
- contraintes connues : urbanisme, incendie, voisinage, paysage, assurance.

## Sortie de l’audit express

L’audit express produit une note courte :

- potentiel faible / moyen / fort ;
- contraintes principales ;
- trajectoire prioritaire ;
- intérêt ou non d’une étude complète ;
- risques évidents ;
- premières hypothèses de valeur locale.

## Sortie de l’audit complet

L’audit complet peut produire :

- état technique ;
- scénarios d’usage ;
- architecture DC-native possible ;
- hypothèse batteries ;
- hypothèse recharge ;
- hypothèse inférence IA ;
- hypothèse coopérative ;
- CAPEX/OPEX estimatifs ;
- garde-fous anti-capture ;
- risques réglementaires ;
- prochaine décision humaine.

## Ce que l’audit ne fait pas

L’audit ne vaut pas autorisation administrative, étude d’exécution, engagement bancaire, certification de sécurité, promesse de rendement ou validation réglementaire finale.

Il sert à décider rationnellement s’il faut continuer.

## Formule

> Avant de transformer un actif, il faut savoir de quoi il dépend, ce qu’il peut produire, ce qu’il peut alimenter, qui peut en bénéficier, et ce qui risque d’être capturé.

Human validation required: yes.
